Plugin chatbot dengan formulir percakapan (tanya-jawab) yang dapat dikustomisasi sepenuhnya dari dasbor WordPress untuk meningkatkan konversi prospek.

== Description ==

Tingkatkan konversi dengan mengubah formulir statis menjadi pengalaman tanya-jawab yang interaktif. "Lead Collector Chatbot" akan menanyakan informasi (nama, email, telepon) satu per satu, membuat pengunjung merasa seperti sedang mengobrol dengan asisten sungguhan.

Semua prospek yang terkumpul disimpan dengan rapi di dalam dasbor WordPress Anda, dan setiap aspek tampilan serta teks chatbot dapat diatur dengan mudah melalui halaman pengaturan.

== Features ==

* [cite_start]**Formulir Percakapan**: Bot akan menanyakan data satu per satu untuk memberikan pengalaman yang lebih alami dan interaktif kepada pengunjung. [cite: 4]
* [cite_start]**Manajemen Prospek (Leads)**: Semua data nama, email, dan telepon yang terkumpul disimpan dan dapat diakses dengan mudah di dalam dasbor WordPress. [cite: 8]
* **Ekspor ke CSV**: Unduh semua data prospek yang terkumpul dalam format file CSV dengan satu kali klik dari halaman manajemen prospek.
* [cite_start]**Kustomisasi Penuh dari Dasbor**: Atur semua aspek chatbot langsung dari halaman pengaturan di WordPress tanpa perlu menyentuh kode. [cite: 7, 21] Fitur kustomisasi meliputi:
    * [cite_start]**Nama Bot & Warna Utama**: Ubah nama bot yang tampil di header dan sesuaikan warna utama chatbot agar cocok dengan desain situs Anda. [cite: 13]
    * **Avatar & Logo**: Unggah gambar untuk avatar bot di header jendela obrolan dan logo untuk gelembung (bubble) chat.
    * [cite_start]**Semua Teks Percakapan**: Personalisasi setiap pesan yang dikirim oleh bot, mulai dari sapaan awal hingga permintaan data. [cite: 13]
    * **Konten Final**: Atur pesan terima kasih yang muncul setelah pengguna mengirimkan data, dan tambahkan tombol/link opsional (misalnya, untuk mengunduh brosur).
* [cite_start]**Personalisasi Pesan**: Gunakan placeholder `{name}` pada pesan untuk menyapa pengunjung secara personal dengan namanya setelah mereka memasukkan data. [cite: 5, 19]
* [cite_start]**Pengaturan Posisi**: Pilih posisi chatbot di sudut Kiri Bawah atau Kanan Bawah layar. [cite: 18]
* [cite_start]**Dukungan Ikon Font Awesome**: Tambahkan ikon dari Font Awesome ke tombol-tombol pilihan untuk tampilan yang lebih menarik. [cite: 6, 20]

== Installation ==

1.  [cite_start]Unggah folder `lead-collector-chatbot` ke direktori `/wp-content/plugins/`. [cite: 9]
2.  [cite_start]Aktifkan plugin melalui menu 'Plugins' di dasbor WordPress. [cite: 9]
3.  [cite_start]**Penting:** Pergi ke menu **Chatbot > Pengaturan** untuk mengatur tampilan, warna, nama, dan semua teks percakapan. [cite: 10]
4.  Selesai! Chatbot siap digunakan.

== Frequently Asked Questions ==

= Bagaimana cara mengubah teks, warna, atau nama bot? =

Sangat mudah. [cite_start]Pergi ke menu **Chatbot > Pengaturan** di dasbor WordPress Anda. [cite: 12] [cite_start]Semua opsi untuk mengubah nama bot, warna utama, avatar, logo, posisi, dan seluruh alur teks percakapan tersedia di halaman tersebut. [cite: 13] [cite_start]Jangan lupa klik "Simpan Perubahan". [cite: 14]

= Bagaimana cara menambahkan ikon ke tombol? =

1.  [cite_start]Kunjungi situs web Font Awesome (fontawesome.com) dan cari ikon gratis yang Anda suka. [cite: 15]
2.  [cite_start]Klik ikon tersebut, lalu salin kelas CSS-nya. [cite: 15] [cite_start]Contoh: `fa-solid fa-comments`. [cite: 16]
3.  [cite_start]Di halaman **Pengaturan** chatbot, tempel kelas tersebut ke kolom "Kelas Ikon Font Awesome" yang sesuai. [cite: 16]
4.  [cite_start]Simpan perubahan Anda. [cite: 17]
